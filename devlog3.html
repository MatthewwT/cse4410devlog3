Olvidar Devlog 3

Abstract puzzle game designed by Mauricio, Antonio, Matthew, & Peter

Olvidar - WIP (Work In Progress)


Leading up to creating the puzzle rooms we ran into a few difficulties, mainly those that were of time management and communication. We had to ensure that everyone was on the same page working towards the same goals; we also had to verify our Unity versions were the same and that we kept the main version of our game on github so everyone could access it. Delays in some code held us back but we were able to overcome it and keep working towards our progress.


As the team had other priorities, the progress halted steeply on the video game design. We discussed many ideas and topics and also researched many tutorials on the side so we can implement it by ourselves while waiting to combine them into one Unity folder. One of these tutorials (introduced by team member Peter) is located at this link here: Build a 2D Platformer Game in Unity


While working on tutorials, we also discussed other items that were not on our radar. This included a menu when the player presses a button (for example the esc button), music design, more art design for progressing levels, and more puzzle ideas that were not too difficult but at the same time not too easy. By the end of this course we might only have the bare bones video game as the sound design and most of the level designs might not be as flushed out as we would like.
